# Data-Analytics-with-Excel-Pivot

This project showcases an interactive sales analytics dashboard built in Microsoft Excel using Pivot Tables, Pivot Charts, and Slicers.

## Features
- Region-wise and Category-wise Sales
- Daily and Product-wise Sales Trends
- Top 3 Products by Revenue
- % Contribution of Each Region
- Interactive Slicers for Region and Category
- Dynamic Column and Line Charts

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Slicers)
- Data Cleaning using Power Query

## Purpose
A hands-on project for practicing data analysis and visualization using Excel.
